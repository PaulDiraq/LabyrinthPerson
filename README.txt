Die Datei die die Main methode dieses Spiels enthält ist
controller/Labyrinth.java.

Daher muss das Spiel via javac controller/Labyrinth.java compiliert werden und
via java controller.Labyrinth aus dem top level Verzeichnis gestartet werden. 


Ziel des Spieles ist es den Spieler (das smiley face) zum Ziel (dem gelben
Rechteck) zu bringen ohne von den Verfolgern (rot) erwischt zu werden.

Wenn das spiel verloren oder gewonnen wurde, kann die Schwierigkeit verändert
werden, indem die Anzahl der Verfolger verändert wird. (Durch eintippen einer
neuen Anzahl an Verfolgern.) Mit <enter> kann ein neues Spiel gestartet
werden. (Momentan ist nur ein Level mit bis zu 3 Verfolgern implementiert.
